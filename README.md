# FJSP-Swarm-Intelligence
本仓库提供《群体智能原理》教材相关算法示例代码，实现了群体智能算法在 **柔性作业车间调度问题（Flexible Job Shop Scheduling Problem, FJSP）** 中的应用。
该项目主要用于 **教学演示、算法学习与实验验证**，帮助理解群体智能算法在复杂调度优化问题中的建模与求解过程。
---

## 项目简介
群体智能（Swarm Intelligence）是一类受自然界群体行为启发的优化方法，这类算法通过简单个体之间的局部交互，实现复杂问题的全局优化能力。

本仓库实现了群体智能算法在 **柔性作业车间调度问题（FJSP）** 中的求解过程，用于展示群体智能算法的基本思想与应用流程。

主要内容包括：
- 调度问题建模
- 群体智能算法实现
- 算法求解流程
- 实验测试示例

---
本仓库适用于：
- 《群体智能原理》课程教学
- 群体智能算法学习
- 调度优化问题实验
- 智能优化算法研究入门
